# UC.SOFTIP Application

## Overview
UC.SOFTIP is a comprehensive web application solution built with ASP.NET Core.

## Project Structure

- **UC.SOFTIP.Api** - RESTful API backend service
- **UC.SOFTIP.Web** - Frontend web application
- **Uc.SignalR.Noti** - Real-time notification service using SignalR
- **DLLs** - Shared libraries and dependencies

## Getting Started

### Prerequisites
- .NET 6.0 SDK or higher
- Visual Studio 2022 or VS Code
- SQL Server (or another supported database)

### Installation

1. Clone the repository
```bash
git clone <repository-url>
cd Project-CI_CD
```

2. Restore dependencies
```bash
dotnet restore
```

3. Build the solution
```bash
dotnet build
```

4. Run the application
```bash
dotnet run --project UC.SOFTIP.Web
```
